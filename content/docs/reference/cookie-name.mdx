# Cookie Name

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `cookie_name` <br/>
**Environment variables:** `COOKIE_NAME` <br/>
**Type:** `string` <br/>
**Default:** `_pomerium` <br/>


**Example:**
```yaml
# config file key
cookie_name: cookie_name

# environment variable
COOKIE_NAME=cookie_name
```

</TabItem>
