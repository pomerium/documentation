# Address

<Tabs>
<TabItem label="Core" value="Core">

---

**Config file key:** `address` <br/>
**Environment variable:** `ADDRESS` <br/>
**Type:** `string` <br/>
**Default:** `:443` <br/>
**Required** <br/>

**Example:**
```yaml
# config file key
address: :8443

# environment variable
ADDRESS=:8443
```
</TabItem>

---

**Address** specifies the Host and Port to serve HTTP requests from. If empty, `:443` is used.

:::tip **Note**

In all-in-one deployments, gRPC traffic will be served on loopback on port `:5443`.

:::
