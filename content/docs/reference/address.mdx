# Address

<Tabs>
<TabItem label="Config file key" value="Config file key">

---

**Name:** `address` <br/>
**Type:** `string` <br/>
**Default:** `:443` <br/>
**Required** <br/>

**Example:**
```yaml title="config.yaml"
address: ":8443"
```

</TabItem>
<TabItem label="Environment variable" value="Environment variable">

---

**Name:** `ADDRESS` <br/>
**Type:** `string` <br/>
**Default:** `:443` <br/>
**Required** <br/>

**Example:**
```yaml title="config.yaml"
ADDRESS=:8443
```

</TabItem>
<TabItem label="Kubernetes" value="Kubernetes">

---

`address` is included in the deployment manifest and not customizable.

</TabItem>
</Tabs>

---

**Address** specifies the Host and Port to serve HTTP requests from. If empty, `:443` is used.

:::tip **Note**

In all-in-one deployments, gRPC traffic will be served on loopback on port `:5443`.

:::
