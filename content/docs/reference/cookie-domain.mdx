# Cookie Domain

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `cookie_domain` <br/>
**Environment variables:** `COOKIE_DOMAIN` <br/>
**Type:** `string` <br/>
**Default:** The host that set the cookie <br/>
**Optional**


**Example:**
```yaml
# config file key
cookie_domain: localhost.pomerium.io

# environment variable
COOKIE_DOMAIN=localhost.pomerium.io
```

</TabItem>

---

**Cookie Domain** sets scope of session cookies issued by Pomerium.

If you specify the domain explicitly, then subdomains would also be included.
