# Autocert Use Staging

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file key:** `autocert_use_staging` <br/>
**Environment variable:** `AUTOCERT_USE_STAGING` <br/>
**Type:** `boolean` <br/>
**Optional** <br/>

**Example:**
```yaml
# config file key
autocert_use_staging: true

# environment variable
AUTOCERT_USE_STAGING=true
```

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

---

**Environment variable:** `AUTOCERT_USE_STAGING` <br/>
**Type:** `boolean` <br/>


</TabItem>

---

Let's Encrypt has strict [usage limits](https://letsencrypt.org/docs/rate-limits/). Enabling the **Autocert Use Staging** setting allows you to use Let's Encrypt's [staging environment](https://letsencrypt.org/docs/staging-environment/), which has more lenient usage limits.
