---

**Config file key:** `autocert_ca` <br/>
**Environment variable:** `AUTOCERT_CA` <br/>
**Type:** `string` <br/>
**Optional** <br/>

**Example:**
```yaml
# config file key
autocert_ca: https://acme.zerossl.com/v2/DV90

# environment variable
AUTOCERT_CA=https://acme.zerossl.com/v2/DV90
```

---

**Autocert CA** is the directory URL of the ACME CA to use when requesting certificates.

:::tip

If set, Autocert CA will override the [**Autocert Use Staging**](/docs/reference/autocert/autocert-use-staging) setting.

:::
