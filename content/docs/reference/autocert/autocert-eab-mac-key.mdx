# Autocert EAB MAC Key

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file key:** `autocert_eab_mac_key` <br/>
**Environment variable:** `AUTOCERT_EAB_MAC_KEY` <br/>
**Type:** `string` <br/>
**Optional** <br/>

**Example:**
```yaml
# config file key
autocert_eab_key_id: EAB_MAC_SECRET_KEY

# environment variable
AUTOCERT_EAB_KEY_ID=EAB_MAC_SECRET_KEY
```

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

---

**Config file key:** `autocert_eab_mac_key` <br/>
**Environment variable:** `AUTOCERT_EAB_MAC_KEY` <br/>
**Type:** `string` <br/>
**Optional** <br/>

**Example:**
```yaml
# config file key
autocert_eab_mac_key: EAB_MAC_Key

# environment variable
AUTOCERT_EAB_MAC_KEY=EAB_MAC_SECRET_KEY
```

</TabItem>
</Tabs>

---

**Autocert EAB MAC Key** is the base64-URL-encoded secret key corresponding to the [Autocert EAB Key ID](/docs/reference/autocert/autocert-eab-key-id).

The Autocert EAB MAC Key setting is required when Autocert EAB Key ID is set.
