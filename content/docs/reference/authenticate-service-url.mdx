# Authenticate Service URL

## Summary

**Authenticate Service URL** is the externally accessible URL for the Authenticate Service. In [split-service mode](/docs/internals/configuration#all-in-one-vs-split-service-mode), this key is required by all services other than Databroker.

## How to configure

| **Type** | **Usage** |
| :--- | :--- |
| `URL` | **required** |

<Tabs>
<TabItem value="Core" label="Core">

| **Config file keys** | **Environment variables** |
| :--- | :--- |
| `authenticate_service_url` | `AUTHENTICATE_SERVICE_URL` |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

| **Config file keys** | **Environment variables** |
| :--- | :--- |
| `authenticate_service_url` | `AUTHENTICATE_SERVICE_URL` |

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

| **Name** |
| :--- |
| `AuthenticateUrl` |


See [Kubernetes `authenticate.url`](/docs/deploying/k8s/reference#authenticate) reference for more information.

</TabItem>
</Tabs>

### Examples

```yaml
# config file key
authenticate_service_url: https://authenticate.corp.example.com

# environment variable
AUTHENTICATE_SERVICE_URL=https://authenticate.corp.example.com

# ingress
authenticate.url: https://authenticate.corp.example.com
```
