# Expiration

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `cookie_expiration` <br/>
**Environment variables:** `COOKIE_EXPIRATION` <br/>
**Type:** `string` <br/>
**Default:** `14h` <br/>


**Example:**
```yaml
# config file key
cookie_expiration: 13h15m0.5s

# environment variable
COOKIE_EXPIRATION=13h15m0.5s
```

</TabItem>

**Cookie Expiration** sets the lifetime of session cookies. After this interval, users must reauthenticate.

See [Go Duration](https://golang.org/pkg/time/#Duration.String) for more information about duration formatting.
