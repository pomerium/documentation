# Databroker Storage Certificate File

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `databroker_storage_cert_file` <br/>
**Environment variables:** `DATABROKER_STORAGE_CERT_FILE` <br/>
**Type:** `string` <br/>
**Optional**

**Example:**
```yaml
databroker_storage_cert_file: /relative/file/location
```

</TabItem>
