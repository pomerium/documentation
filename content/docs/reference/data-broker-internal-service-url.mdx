# Databroker Internal Service URL

## Summary

**Databroker Internal Service URL** overrides [`databroker_service_url`](/docs/reference/data-broker-service-url) when determining the TLS Certificate for the Databroker service to listen with.

## How to configure

| **Type** | **Default**                                  |
| :------- | :------------------------------------------- |
| `URL`    | `http://localhost:5443` (In all-in-one mode) |

<Tabs>
<TabItem value="Core" label="Core">

| **Config file keys**               | **Environment variables**          |
| :--------------------------------- | :--------------------------------- |
| `databroker_internal_service_url`  | `DATABROKER_INTERNAL_SERVICE_URL`  |
| `databroker_internal_service_urls` | `DATABROKER_INTERNAL_SERVICE_URLS` |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

`databroker_internal_service_url` and `databroker_internal_service_urls` are bootstrap configuration settings and are not configurable in the Console.

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

`databroker_internal_service_url` is not customizable in Kubernetes

</TabItem>
</Tabs>

### Examples

```yaml
databroker_internal_service_urls:
              - http://localhost:5443
              - http://service_url.com

DATABROKER_INTERNAL_SERVICE_URL=http://localhost:5443
```
