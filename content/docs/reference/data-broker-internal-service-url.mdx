# Databroker Internal Service URL

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `databroker_internal_service_url` or `databroker_internal_service_urls` <br/>
**Environment variables:** `DATABROKER_INTERNAL_SERVICE_URL` or `DATABROKER_INTERNAL_SERVICE_URLS` <br/>
**Type:** `URL` <br/>
**Default:** `http://localhost:5443` (In all-in-one mode)

**Example:**
```yaml
databroker_internal_service_url: http://localhost:5443
```

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

---

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

---

`databroker_internal_service_url` is not customizable in Kubernetes

</TabItem>
</Tabs>

---

**Databroker Internal Service URL** overrides [`databroker_service_url`](/docs/reference/data-broker-service-url) when determining the TLS Certificate for the Databroker service to listen with.
