# Databroker Storage Certificate Authority

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `databroker_storage_ca_file` <br/>
**Environment variables:** `DATABROKER_STORAGE_CA_FILE` <br/>
**Type:** `string` <br/>
**Optional**

**Example:**
```yaml
databroker_storage_ca_file: /relative/file/location
```
