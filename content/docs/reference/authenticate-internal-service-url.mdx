# Authenticate Internal Service URL

<Tabs>
<TabItem label="Core" value="Core">

---

**Config file key:** `authenticate_internal_service_url` <br/>
**Environment variable:** ``AUTHENTICATE_INTERNAL_SERVICE_URL`` <br/>
**Type:** `URL` <br/>
**Required** (In [split-service mode](/docs/internals/configuration#all-in-one-vs-split-service-mode) only)

**Example:**

```yaml
# config file key
authenticate_internal_service_url: https://authenticate.internal

# environment variable
AUTHENTICATE_INTERNAL_SERVICE_URL=https://authenticate.internal
```

</TabItem>
