# Cookie Secret

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `cookie_secret` <br/>
**Environment variables:** `COOKIE_SECRET` <br/>
**Type:** `string` <br/>
**Optional**


**Example:**
```yaml
cookie_secret: tdkuWzUelRukP/6VYzopfh6kis7y5u5Ldl3MrIq9ZR0=
```

To generate a random key, run the following command:

```shell
head -c32 /dev/urandom | base64
```
