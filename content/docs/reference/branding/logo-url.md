# Logo URL

## Summary

**Logo URL** customizes the logo displayed in the Enterprise Console and Open Source endpoints.

## How to configure

| **Type** | **Default**   |
| :------- | :------------ |
| `URL`    | Pomerium Logo |

See [Custom Branding / Errors](/docs/capabilities/branding) for more information.
