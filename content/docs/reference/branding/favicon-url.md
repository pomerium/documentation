# Favicon URL

## Summary

**Favicon URL** customizes the Favicon displayed in the Enterprise Console and Open Source endpoints.

## How to configure

| **Type** | **Default**      |
| :------- | :--------------- |
| `URL`    | Pomerium Favicon |

See [Custom Branding / Errors](/docs/capabilities/branding) for more information.
