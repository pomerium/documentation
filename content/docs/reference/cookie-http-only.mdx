# Cookie HTTP Only

## Summary

If true, **HTTP Only Cookie** forbids JavaScript from accessing the cookie.

## How to configure

| **Type** | **Default** |
| :--- | :--- |
| `boolean` | `true` |

<Tabs>
<TabItem value="Core" label="Core">

| **Config file keys** | **Environment variables** |
| :--- | :--- |
| `cookie_http_only` | `COOKIE_HTTP_ONLY` |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

Configure **Cookie HTTP Only** with the **Javascript Security** toggle button in the Console. The button has three states:
- **Unset** ("-") uses the value in your configuration file
- **Checkmark** sets `cookie_http_only` to `true`
- **Empty** sets `cookie_http_only` to `false`

![Configure Cookie HTTP only in the Console](./img/cookies/cookie-settings.png)

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

| **Name** |
| :--- |
| `cookie.httpOnly` |

See Kubernetes [Cookie Reference](/docs/deploying/k8s/reference#cookie) for more information.

</TabItem>
</Tabs>

### Examples

```yaml
# config file key
cookie_http_only: false

# environment variable
COOKIE_HTTP_ONLY=false

# ingress
cookie.httpOnly: false
```
