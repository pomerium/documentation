# HTTP Only Cookie

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `cookie_http_only` <br/>
**Environment variables:** `COOKIE_HTTP_ONLY` <br/>
**Type:** `boolean` <br/>
**Default:** `true` <br/>


**Example:**
```yaml
# config file key
cookie_http_only: false

# environment variable
COOKIE_HTTP_ONLY=false
```

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

---

**Config file keys:** `cookie_http_only` <br/>
**Environment variables:** `COOKIE_HTTP_ONLY` <br/>
**Type:** `boolean` <br/>
**Default:** `true` <br/>

In the Enterprise Console, configure `cookie_http_only` with the **Javascript Security** toggle button. The button has three states:
- **Unset** ("-") uses the value in your `config.yaml` file
- **Checkmark** sets `cookie_http_only` to `true`
- **Empty** sets `cookie_http_only` to `false`

<TabItem value="Kubernetes" label="Kubernetes">

---

**Name:** `cookie.expiration` <br/>
**Default:** `14h` <br/>
**Type:** `string`

**Example:**
```yaml
cookie.httpOnly: false
```

See Kubernetes [Cookie Reference](/docs/deploying/k8s/reference#cookie) for more information.
