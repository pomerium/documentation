# Data Broker Storage TLS Skip Verify

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `databroker_storage_tls_skip_verify` <br/>
**Environment variables:** `DATABROKER_STORAGE_TLS_SKIP_VERIFY` <br/>
**Type:** `string` <br/>
**Optional**

**Example:**

```yaml
databroker_storage_skip_verify: /relative/file/location
```

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

---

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

---

</TabItem>
</Tabs>

---

If **Databroker Storage TLS Skip Verify** is set, the TLS connection to the storage backend will not be verified.
