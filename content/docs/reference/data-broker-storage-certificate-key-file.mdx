# Databroker Storage Certificate Key File

<Tabs>
<TabItem value="Core" label="Core">

---

**Config file keys:** `databroker_storage_key_file` <br/>
**Environment variables:** `DATABROKER_STORAGE_KEY_FILE` <br/>
**Type:** `string` <br/>
**Optional**

**Example:**
```yaml
databroker_storage_key_file: /relative/file/location
```

</TabItem>
