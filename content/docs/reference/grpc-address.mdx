# gRPC Address

## Summary

**gRPC Address** specifies the host and port to serve gRPC requests from.

## How to configure

| **Type** | **Default** |
| :-- | :-- |
| `string` | `:443` (`:5443` if in [all-in-one](/docs/internals/configuration#all-in-one-vs-split-service-mode) mode) |

<Tabs>
<TabItem value="Core" label="Core">

| **Config file keys** | **Environment variables** |
| :------------------- | :------------------------ |
| `grpc_address`       | `GRPC_ADDRESS`            |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

`grpc_address` is a bootstrap configuration setting and is not configurable in the Console.

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

**gRPC Address** is not customizable with Kubernetes

</TabItem>
</Tabs>

### Examples

```yaml
# config file key
grpc_address: :8443

# environment variable
GRPC_ADDRESS=:8443
```
