# Cookie Name

- Environmental Variable: `COOKIE_NAME`
- Config File Key: `cookie_name`
- Kubernetes: [`cookie.name`](/docs/deploy/k8s/reference#cookie)
- Type: `string`
- Default: `_pomerium`

The name of the session cookie sent to clients.
