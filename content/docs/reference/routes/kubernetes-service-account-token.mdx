# Kubernetes Service Account Token

## Summary

**Kubernetes Service Account Token** authenticates requests to a Kubernetes API server.

Pomerium will [impersonate](https://kubernetes.io/docs/reference/access-authn-authz/authentication/#user-impersonation) the Pomerium user's identity, and Kubernetes RBAC can be applied to identity provider users and groups.

## How to configure

<Tabs>
<TabItem value="Enterprise" label="Enterprise">

Set the **Kubernetes Service Account Token** under **General** route settings in the Console:

![Set k8s service account token in the Console](../img/routes/k8s/kubernetes-bearer-token.png)

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

| **YAML**/**JSON** settings | **Type** | **Usage** |
| :-- | :-- | :-- |
| `kubernetes_service_account_token` | `string` | **optional** |
| `kubernetes_service_account_token_file` | `bearer token` file path | **optional** |

### Examples

```yaml
ingress.pomerium.io/kubernetes_service_account_token: eyJ0eXAiOiJKV1QiLCJhbGciOiJ...

ingress.pomerium.io/kubernetes_service_account_token_file: /var/run/secrets/kubernetes.io/serviceaccount/token
```
