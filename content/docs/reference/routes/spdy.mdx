# SPDY

## Summary

If set, **SPDY** enables proxying of SPDY protocol upgrades.

## How to configure

| **Type** | **Default** |
| :--- | :--- |
| `boolean` | `false` |

<Tabs>
<TabItem value="Core" label="Core">

| **YAML**/**JSON** setting |
| :--- |
| `allow_spdy` |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

Enable **SPDY** in the Console:

![Enable SPDY in the console](../img/routes/timeouts/spdy.png)

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

| **Name** |
| :--- |
| `allow_spdy` |

See Kubernetes [Ingress](/docs/deploying/k8s/ingress#annotations) for information

</TabItem>
</Tabs>

### Examples

```YAML
allow_spdy: true
```
