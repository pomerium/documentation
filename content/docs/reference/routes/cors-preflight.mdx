# CORS Preflight

## Summary

**CORS Prefligh** allows unauthenticated HTTP OPTIONS requests as [per the CORS spec](https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS#Preflighted_requests).

## How to configure

| **Type** | **Default** | **Usage** |
| :--- | :--- | :--- |
| `boolean` | `false` | **optional** |

<Tabs>
<TabItem value="Core" label="Core">

| **YAML**/**JSON** setting |
| :--- |
| `cors_preflight` |

</TabItem>
<TabItem value="Enterprise" label="Enterprise">

Enable **CORS Preflight** in the **Policy Builder** in the Console:

![enable cors preflight in the console](../img/routes/cors/cors-preflight.png)

</TabItem>
<TabItem value="Kubernetes" label="Kubernetes">

**CORS Preflight** is not customizable with Kubernetes.

</TabItem>
</Tabs>

### Examples

```yaml
cors_preflight: true
```
